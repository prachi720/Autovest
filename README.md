# AutoVest 🤖💰
**🌐 Live Demo:** [Click here to view the live app!](https://autovest-app.vercel.app/)           

AutoVest is a smart, automated micro-investing prototype designed to simplify and automate small-scale investing based on intelligent spending and financial logic. 

## 🚀 Technologies Used

This frontend application is built using a modern, fast, and optimized tech stack:
*   **Vite** – Next-generation frontend tooling for rapid development.
*   **React (v18)** – Component-based UI rendering[cite: 16, 19].
*   **TypeScript** – Static type checking for robust, scalable code[cite: 19].
*   **Tailwind CSS** – Utility-first CSS framework for custom responsive styling[cite: 19].
*   **shadcn/ui** – Beautifully designed, accessible UI components[cite: 19].
*   **Framer Motion** – Smooth, interactive fluid animations.
*   **Lucide React** – Clean, modern iconography.


## 💻 Local Development Setup

To get a copy of this project up and running on your local machine, ensure you have **Node.js** installed, then follow these simple steps:

### 1. Clone the Repository
```bash
git clone https://github.com/prachi720/Autovest-frontend.git
cd Autovest-frontend
```
### 2. Install Dependencies
Recreate the node_modules folder and install all locked library versions securely:

```Bash
npm install
# OR if you prefer Bun:
bun install
```

### 3. Run the Development Server
Launch the local development environment with instant hot-reloading[cite: 19]:

```Bash
npm run dev
```

Once started, open your browser and navigate to the local server address (usually http://localhost:5173) to view the application.

### Available Build ScriptsInside the package.json file, the following automation scripts are configured:  
```npm run dev``` – Starts the Vite dev server.  
```npm run build ```– Bundles the application into highly optimized static assets for production deployment.  
```npm run lint``` – Runs ESLint to check your codebase for errors and structural issues[cite: 16].        
```npm run test``` – Executes the testing suite using Vitest[cite: 16].
